# 🌐 Anime Web Analytics: Live Data Integration

## 🧠 Project Overview

- This Power BI project provides a comprehensive analysis of the Anilab app ecosystem and broader anime trends.
- Unlike traditional projects that use static datasets, this dashboard is powered by live data fetched directly from web sources (MyAnimeList).

## 🛠️ Data Engineering Process (The "Magic" Behind the Scenes)

o The core strength of this project is the ETL Pipeline:

- Extraction: Utilized Power BI's Web Connector to scrape/fetch raw data from web servers.

- Transformation: Used Power Query to handle data cleaning, converting raw web text into structured formats, handling null values, and formatting dates/ratings.

- Loading: Designed a Star Schema to optimize dashboard performance.

## 📊 Key Analytics & Features

- App Analysis: Insights into the Anilab application performance and user metrics.

- Live Trend Monitoring: Real-time ratings and popularity metrics fetched from the web.

- Interactive Visuals: - Genre Distribution: Identifying market saturation and popular themes.

- Rating vs. Popularity: Correlation analysis between critical acclaim and fan following.

- Advanced Slicers: Filtering by status (Finished/Airing), Type (TV/Movie/OVA), and more.
